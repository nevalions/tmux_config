# tmux config
```
    cd
    unzip tmux_config-main.zip
    mv tmux_config-main/.tmux.conf.local ~/.tmux.conf.local
    mv tmux_config-main/.tmux.conf ~/.tmux.conf
    rm tmux_config-main.zip
    rm -rf tmux_config-main
```
